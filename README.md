Clipboard Manager Pro - Chrome Extension
A lightweight, privacy-first clipboard manager for Chrome that saves your copied text/links, lets you pin important items, and unlocks unlimited records with Pro activation.

✨ Key Features
Free Version (Forever Free)
Save up to 3 clipboard records (text/links)
Pin important items to keep them at the top
One-click copy/paste from history
Edit, delete, and search clipboard entries
No account required, all data stored locally
No ads, no tracking, 100% privacy-focused
Pro Version (Unlock Full Potential)
Unlimited clipboard history (no record limits)
Advanced search with keyword filtering
Bulk delete/export functionality
Cloud sync across devices (coming soon)
Priority support and feature updates
📥 Installation Instructions (No Chrome Web Store Required)
Method 1: Install from Unpacked Files (Recommended)
Download the latest dist.zip from the Releases page (or main branch)
Extract the ZIP file to a folder on your computer (keep this folder - do not delete it)
Open Chrome browser and go to chrome://extensions/
Enable Developer mode (toggle switch in the top-right corner)
Click Load unpacked (top-left corner)
Select the extracted dist folder
The extension icon will appear in your Chrome toolbar - click it to start using!
Method 2: Install CRX File (Alternative)
Download clipboard-manager-pro.crx from the repository
Open Chrome and go to chrome://extensions/
Enable Developer mode (top-right)
Drag and drop the .crx file directly into the extensions page
Click "Add extension" when prompted
⚠️ Important Note: Chrome will show a "Developer mode extension" warning - this is normal for non-Web Store extensions. The extension only accesses your local clipboard data and does not send data to any servers.
🚀 How to Use
Basic Usage
Click the extension icon in your Chrome toolbar to open the interface
Copy any text/link - it will automatically be added to your clipboard history
View your history in the scrollable list (3 entries max for free version)
Use the action buttons on each record:
📌 Pin: Keep important entries at the top of your history
✏️ Edit: Modify the content of any saved record
📋 Copy: Copy the record back to your clipboard (shows "Copied" confirmation)
🗑️ Delete: Remove individual records from history
Advanced Actions
Search: Type in the search box to filter records by keyword (real-time filtering)
Clear All: Click "Clear All Records" (bottom-left) to delete all history (irreversible)

🔧 Troubleshooting
Extension Not Loading
Ensure you extracted the full ZIP file (check for the manifest.json file in the dist folder)
Verify Developer mode is enabled in chrome://extensions/
Do not move/delete the extracted dist folder after installation
Refresh the extensions page (chrome://extensions/) and click "Refresh" on the extension
Clipboard Not Saving
Make sure the extension has permission to access your clipboard (Chrome prompts for this on first use)
Check that you're not copying duplicate content (the extension skips exact duplicates to save space)
For free version: Ensure you haven't exceeded the 3-record limit (delete old records to add new ones)
Pro Activation Issues
Double-check your activation code (case-insensitive, spaces/hyphens are ignored)
Verify you're using a valid Pro code (test codes work for evaluation purposes)
If activation fails: Clear extension storage (in chrome://extensions/ → Details → Storage → Clear storage) and try again

❓ Frequently Asked Questions (FAQ)
Q: Is my clipboard data safe?
A: Yes! All data is stored locally in Chrome's extension storage - we never access or transmit your clipboard data.
Q: What happens if I delete the extracted dist folder?
A: The extension will stop working. Keep the folder in a permanent location (e.g., Documents/ClipboardManagerPro).
Q: Can I use this on other Chromium browsers (Edge, Brave, etc.)?
A: Yes! The extension works on all Chromium-based browsers - follow the same installation steps.
Q: How do I update the extension?
A: Download the latest dist.zip, delete the old extension from chrome://extensions/, and reinstall using the new files.
